Text Content

In this section, you'll learn about various text-related HTML tags and their usage.

Headings

Headings range from <h1> to <h6>. Each heading tag represents a different level of section heading.

Paragraphs

The <p> tag is used to define blocks of text.

Blockquotes

Use <blockquote> for quoted sections of text.

Preformatted Text

The <pre> tag is used for text with preserved whitespace and formatting.

Bold, Italic, Underline, Strikethrough

Use <b> or <strong> for bold text, <i> or <em> for italic text, <u> for underline, and <s> for strikethrough.

Subscript and Superscript

Use <sub> for subscript text and <sup> for superscript text.

Abbreviations, Citations, and Inline Quotes

Use <abbr> for abbreviations, <cite> for citations, and <q> for inline quotes.

Images and Media

Embedding Images

Embedding images using the <img> tag. Example: <img src="image.jpg" alt="Description">.

Figure and Figcaption

Use <figure> to group images with a caption and <figcaption> to add a caption. Example: <figure><img src="image.jpg" alt="Description"><figcaption>Caption text</figcaption></figure>.

Embedding Audio

Embedding audio using the <audio> tag. Example: <audio controls><source src="audio.mp3" type="audio/mpeg">Your browser does not support the audio element.</audio>.

Embedding Video

Embedding video using the <video> tag. Example: <video controls><source src="video.mp4" type="video/mp4">Your browser does not support the video tag.</video>.

Links and Navigation

Anchor Links

Anchor links using the <a> tag. Example: <a href="https://example.com">Example</a>.

Navigation Using <nav>

Navigation using the <nav> tag. Example: <nav><ul><li><a href="home.html">Home</a></li><li><a href="about.html">About</a></li></ul></nav>.

Linking to Other Pages

Linking to other pages within the documentation using relative URLs. Example: <a href="page2.html">Page 2</a>.

Email and Telephone Links

Email links using mailto:. Example: <a href="mailto:someone@example.com">Email Us</a>.

Telephone links using tel:. Example: <a href="tel:+1234567890">Call Us</a>.

Forms

Form Structure

Form structure using <form>, <fieldset>, and <legend>. Example: <form action="submit.html" method="post"><fieldset><legend>Personal Information</legend><label for="name">Name:</label><input type="text" id="name" name="name"></fieldset></form>.

Input Types

Different input types such as <input type="text">, <input type="password">, etc. Example: <input type="email" id="email" name="email">.

Textarea, Select, and Option

Using <textarea>, <select>, and <option>. Example: <select name="options"><option value="1">Option 1</option><option value="2">Option 2</option></select>.

Buttons

Using <button>, <input type="submit">, and <input type="reset">. Example: <button type="submit">Submit</button>.

Tables

Basic Table Structure

Basic table structure using <table>, <tr>, <th>, and <td>. Example: <table><tr><th>Header 1</th><th>Header 2</th></tr><tr><td>Data 1</td><td>Data 2</td></tr></table>.

Table Headers and Captions

Table headers using <thead>, <tbody>, and <tfoot>. Example: <table><thead><tr><th>Header 1</th><th>Header 2</th></tr></thead><tbody><tr><td>Data 1</td><td>Data 2</td></tr></tbody><tfoot><tr><td>Footer 1</td><td>Footer 2</td></tr></tfoot></table>.

Colspan and Rowspan

Using colspan and rowspan attributes for spanning cells across columns or rows. Example: <td colspan="2">Spanning Cell</td>.

Responsive Tables

Making tables responsive using CSS techniques like overflow and media queries.

Lists

Unordered Lists

Unordered lists using <ul> and <li>. Example: <ul><li>Item 1</li><li>Item 2</li><li>Item 3</li></ul>.

Ordered Lists

Ordered lists using <ol> and <li>. Example: <ol><li>First</li><li>Second</li><li>Third</li></ol>.

Definition Lists

Definition lists using <dl>, <dt>, and <dd>. Example: <dl><dt>Term</dt><dd>Definition</dd></dl>.

Nested Lists

Nesting lists by placing <ul> or <ol> inside <li> elements.

Metadata

Meta Tags

Meta tags within the <head> section to provide metadata about the document. Example: <meta charset="UTF-8">.

Title Tag

The <title> tag to define the document's title. Example: <title>Document Title</title>.

Link Tag

The <link> tag to link external resources like stylesheets. Example: <link rel="stylesheet" href="styles.css">.

Base Tag

The <base> tag to specify the base URL for relative URLs in the document. Example: <base href="https://example.com/">.

Miscellaneous Tags

Comments

HTML comments using <!-- --> to add notes or explanations within the code. Example: <!-- This is a comment -->.

Div and Span

The <div> and <span> tags for grouping and styling purposes. Example: <div>Content here</div>.

Section, Article, and Aside

The <section>, <article>, and <aside> tags for defining sections of content. Example: <section>Section content</section>.

Footer and Header

The <footer> and <header> tags for defining the footer and header of a document or section. Example: <footer>Footer content</footer>.

Progress and Meter

The <progress> and <meter> tags for displaying progress and measurement. Example: <progress value="70" max="100">70%</progress>.
